# OVK HUD

## Note: This mod is in an early stage of development.

## Features:
   - Health Counter
   - Zombie Counter

## How to use this mod?

1. Extract the .zip
   
2. Open folder "OVK_HUD"

3. copy the "zm" folder to "AppData\Local\Plutonium\storage\t6\scripts\zm\"

4. start a game
